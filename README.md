# Markdown在线编辑器

本项目为字节跳动青训营MiniMarkDownEditor项目。
我们旨在开发一个支持 Markdown 编写和实时预览的编辑器，帮助用户更直观地完成内容创作，同时为学习和实践前端开发技术提供一个实际案例。

## 如何实现

在左侧输入框中写入文本后，markdown解析器将会把markdown文本解析成html元素，并在右侧结果框输出。
不需要后端服务器。

## 使用说明

安装依赖库
```
npm install
```

运行
```
npm run dev
```

单元测试
```
npm test
```

端到端测试
```
npm run cypress:open
```
